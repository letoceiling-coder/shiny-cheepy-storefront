# Cheepy Storefront

Маркетплейс модной одежды и аксессуаров.

## Технологии

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Разработка

```sh
npm i
npm run dev
```

## Сборка

```sh
npm run build
```

## Деплой

Соберите проект и загрузите содержимое папки `dist/` на сервер в директорию сайта.
